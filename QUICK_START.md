# 🚀 快速开始指南

## 第一步：获取 DeepSeek API Key

1. 访问 DeepSeek 官网：https://platform.deepseek.com/
2. 注册/登录账号
3. 进入 API Keys 页面
4. 点击「Create API Key」创建新的密钥
5. 复制生成的 API Key（请妥善保管）

## 第二步：配置 API Key

编辑项目根目录的 `.env.local` 文件：

```env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

将 `sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` 替换为你的真实 API Key。

## 第三步：启动项目

项目已经启动！访问：

- **本地访问**: http://localhost:3000
- **局域网访问**: http://192.168.101.71:3000

## 📱 使用流程

### 1. 分析股市图表

1. 打开首页 http://localhost:3000
2. 点击上传区域，选择股市K线图（支持 PNG, JPG, JPEG）
3. （可选）在文本框输入补充描述，例如：
   ```
   大盘今日震荡，成交量萎缩，MACD即将金叉
   ```
4. 点击「🤖 AI 分析」按钮
5. 等待 5-10 秒，查看分析结果

**分析结果包括：**
- 📍 支撑位 / 阻力位
- 📈 操作方向（做多/做空/观望）
- ⚠️ 止损位
- 🎯 目标位
- 💡 详细分析逻辑

### 2. 每日复盘

**何时复盘？**
- 工作日收盘后（15:00 之后）
- 对比AI预测与实际走势

**如何复盘？**

1. 访问「历史记录」页面
2. 找到状态为「待复盘」的分析记录
3. 点击「开始复盘」按钮
4. 填写实际数据：
   - 实际最高价
   - 实际最低价
   - 实际收盘价
   - （可选）复盘总结
5. 点击「提交复盘」
6. 系统自动计算准确率并保存

### 3. 查看历史

历史记录页面显示所有分析，包括：
- ✅ 已复盘：绿色标签，显示准确率
- ⏳ 待复盘：黄色标签，等待收盘后复盘

## 🧠 持续学习原理

每次新分析时，系统会自动：

1. 读取最近 5 条已复盘的分析记录
2. 整理成「预测 vs 实际」对比数据
3. 作为历史经验传递给 DeepSeek AI
4. AI 参考过往成功/失败案例，优化预测

**复盘越多，AI 越准！**

## ⚙️ 常见问题

### Q: API 调用失败怎么办？

**A:** 检查以下几点：
1. `.env.local` 中的 API Key 是否正确
2. DeepSeek 账户余额是否充足
3. 网络连接是否正常
4. 查看控制台错误信息

### Q: 图片上传后没有反应？

**A:**
1. 检查图片格式（支持 PNG, JPG, JPEG）
2. 图片大小建议小于 5MB
3. 打开浏览器开发者工具查看错误

### Q: 准确率如何计算？

**A:** 简单算法（可自行优化）：
- 方向正确 + 价格接近目标 = 70-100分
- 方向正确但价格偏差大 = 40-70分
- 方向错误 = 0-30分

### Q: 数据存储在哪里？

**A:**
- 分析记录：`data/analyses.json`
- 复盘记录：`data/reviews.json`
- 上传图片：`public/uploads/`

### Q: 可以处理多少条数据？

**A:**
- JSON 文件适合 < 10,000 条记录
- 超过建议迁移到数据库（PostgreSQL、MySQL等）
